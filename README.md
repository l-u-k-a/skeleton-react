# skeleton-react
reactjs basic boilerplate
## what's in?
  * Webpack 2
  * HMR
  * SCSS
  * React

## Manual
Clone the repository
```sh
git clone https://github.com/l-u-k-a/skeleton-react.git
```
Install dependencies
```sh
npm install
```

Run
```sh
npm start
```

Lint
```sh
npm run lint
```

Build
```sh
npm run build
```
